A friendly, modern one-page starter website for Andrea Scheppler’s pediatric SLP private practice.

## Files
- `index.html` — homepage content and structure
- `styles.css` — responsive visual design
- `andrea.jpg` — add Andrea’s portrait here before preview/deploy

## Image setup
Copy the attached portrait into this folder and name it:

```text

```
